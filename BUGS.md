# /entry/add should respect the date it's in
The fab button should open a dialog with class .sheet (fullscreen).
If a day in the past is selected, it should display a warning.
So I'm proposing a hidden input field that contains the entry_date.

# Entries text should be editable
All entries text, both note field and transcribed field should be editable.
Trackables attached to an entry should be deletable. 

# Sensitive filter
There should be a toggle at the top of entries that toggles display of private/sensitive entries and trackables.

# Compact view
There should be a view switch at the top of entries to display a compact view, or default (like it is now)
Compact view should put everything into one "box".
